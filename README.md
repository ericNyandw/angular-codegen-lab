## 🛠️ Installation & Démarrage rapide

Comme le code généré est un artefact de build, il est exclu du dépôt Git. Suivez ces étapes pour initialiser le projet localement :

1. **Installer les dépendances** :
   ```bash
   npm install
   ```
2. **Restaurer les services API (Génération)**:
Exécutez les scripts définis dans le package.json pour créer le dossier src/app/api/ :
```bash
npm start 
```
Rendez-vous sur http://localhost:4200 pour tester l'interface réactive.
